## Kurulum & Çalıştırma

### 1. Gereksinimler

- Node.js ≥ 18
- Google Gemini API Key (ücretsiz almak için: https://aistudio.google.com/app/apikeys)

### 2. Proje Kurulumu

```bash
# Repoyu klonla
git clone https://github.com/bcila/devocase.git
cd text-to-flowchart

# Paketleri yükle
npm install

# .env.local dosyası oluştur ve Gemini API key'ini ekle
cp .env.example .env.local
```
